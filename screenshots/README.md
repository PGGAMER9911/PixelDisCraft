# Screenshots

This folder contains screenshots demonstrating PixelDisCraft in action.

## Required Screenshots

### `chat-bridge.png`
Shows the bi-directional chat bridge:
- A Minecraft player sending a message that appears in Discord
- A Discord user sending a message that appears in Minecraft

### `discord-command.png`
Shows Discord slash commands in action:
- An admin using `/server` to view server status
- An admin using `/mc` to execute a Minecraft command

---

## How to Add Screenshots

1. Take a screenshot in Discord showing the feature
2. Save it as a PNG file in this folder
3. Use the exact filenames above so the README embeds them correctly

> **Tip:** Use Discord's screenshot feature (Print Screen / Snipping Tool) and crop to show just the relevant channel content.
